# Roadmap

This is a living document reflecting the planned direction for StackMCP. Priorities and timelines may shift based on feedback and development progress.

## Current focus

- **Core daemon** — building the local MCP server that manages tool connections and handles requests
- **Tool integrations** — developing connectors for common developer tools, databases, and APIs
- **Workflow engine** — implementing the JSON-based workflow runner

## Planned

| Area | Description |
|------|-------------|
| Community templates | Curated library of workflow templates contributed by the community |
| Documentation | More examples and guides for common workflows |
| Tool SDK | Interfaces for writing custom tool connectors |

## Not planned

- Hosted/SaaS version of StackMCP
- Removing or forking the MCP protocol
- Windows support (currently focused on macOS)

## Contributing

StackMCP is open source. If you have ideas, feature requests, or want to contribute, see [CONTRIBUTING.md](CONTRIBUTING.md).
